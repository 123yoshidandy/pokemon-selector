# ポケモン選出サポートツール - 使い方ガイド

## サーバー起動

```bash
python3 -m http.server 8080
```

## アプリケーションアクセス

### メインアプリケーション
http://localhost:8080/public/index.html

## 実装した機能

### 1. 推奨選出表示（自分の6匹から3匹を選ぶ）
- 自分と相手のパーティを入力
- 「推奨選出を表示」ボタンをクリック
- タイプ相性を考慮した最適な3匹の組み合わせを提案

### 2. 自動補完機能
- ポケモン名入力欄にフォーカス
- 文字を入力すると候補リストが表示される

### 3. ランキング上位ポケモン表示
- 「ランキング上位を表示」ボタンをクリック
- 選出率上位30体をカード形式で表示
- カードクリックで自分のパーティに追加

### 4. 相性分析
- パーティを入力して「相性分析」ボタンをクリック
- タイプ相性に基づくスコア計算
- 色分け表示（緑：有利、赤：不利）

### 5. 分析サマリー
- 総合スコア表示
- 推奨選出（自分のパーティから上位3体）
- 警戒すべきポケモン（相手のパーティから上位3体）

## ブラウザコンソールでの確認

開発者ツールのコンソールで以下を確認：

```javascript
// 関数が正しく読み込まれているか
console.log(typeof window.buttonClick);  // "function"
console.log(typeof window.showRecommendations);  // "function"

// 手動で推奨選出を表示
window.showRecommendations();
```

## トラブルシューティング

### エラーが出る場合
1. サーバーがプロジェクトルートから起動されているか確認
2. ブラウザのキャッシュをクリア（Ctrl+Shift+R または Cmd+Shift+R）
3. コンソールでエラーメッセージを確認

### データが読み込まれない場合
1. simple-test.htmlでデータファイルの読み込み状況を確認
2. ネットワークタブで404エラーがないか確認

## 初期値

### 自分のパーティ（デフォルト）
- マスカーニャ、ラウドボーン、ウェーニバル
- フシギバナ、リザードン、カメックス

### 相手のパーティ（デフォルト）
- 選出率上位6体：ディンルー、パオジアン、コライドン、バドレックス、ミライドン、ハバタクカミ
