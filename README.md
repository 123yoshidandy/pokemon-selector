# ポケモン対戦支援ツール

ポケモンSVランクマッチの勝率向上を支援する総合分析ツール
https://123yoshidandy.github.io/pokemon-selector/

## 📌 概要

Pokemon HOMEの公式データと独自アルゴリズムを組み合わせ、対戦における戦略的な意思決定を支援する包括的なWebアプリケーションです。

## ✨ 主要機能

### 1. 📊 構築支援
- **タイプ相性分析**: パーティ全体のタイプバランスを可視化
- **環境メタ対策**: 使用率上位ポケモンへの対策度を評価
- **相性補完提案**: 既存メンバーと相性の良いポケモンを推薦
- **弱点カバー分析**: パーティの弱点を特定し改善案を提示

### 2. 🎯 選出支援
- **対戦相手分析**: 相手パーティ6体の脅威度を数値化
- **最適選出計算**: 勝率を最大化する3体の組み合わせを提案
- **相手選出予測**: 相手が選出する可能性の高いポケモンを予測
- **キーマッチアップ特定**: 勝敗を左右する重要な対面を明示

### 3. ⚔️ 立ち回り支援
- **リアルタイム戦術提案**: 現在の盤面から最適な行動を提示
- **交代タイミング判定**: 有利な交代機会を検出
- **ダメージ予測**: 次ターンの被ダメージを予測
- **勝利条件の明確化**: 勝利への道筋を具体的に提示

### 4. 🧮 ダメージ計算
- **詳細ダメージ計算**: 努力値・個体値・性格補正を含む精密計算
- **環境効果対応**: 天候・フィールド・状態異常の影響を反映
- **確定数表示**: 何発で倒せるか/倒されるかを明示
- **ポケモン自由入力**: データベース外のポケモンも計算可能

### 5. 🏆 使用率ランキング
- **リアルタイムランキング**: シングル/ダブルの最新使用率
- **詳細統計データ**: 技・特性・持ち物・テラスタイプの使用率
- **トレンド分析**: 前回からの順位変動を追跡
- **メタゲーム分析**: 環境の変化を可視化

## 🔄 自動データ更新

### Pokemon HOME連携
GitHub Actionsにより毎日自動更新されるデータ：

| データ種別 | 更新頻度 | 件数 | 内容 |
|---------|---------|------|------|
| 使用率ランキング | 毎日 | 30位まで | シングル/ダブルバトル |
| 技構成 | 毎日 | 6,000件以上 | 各ポケモンの採用技 |
| 特性分布 | 毎日 | 1,000件以上 | 特性の使用率 |
| 持ち物 | 毎日 | 3,600件以上 | アイテム採用率 |
| テラスタイプ | 毎日 | 2,800件以上 | テラスタイプ分布 |

## 🚀 クイックスタート

### オンラインで使用
```
https://123yoshidandy.github.io/pokemon-selector/
```

### ローカル環境で実行
```bash
# リポジトリのクローン
git clone https://github.com/123yoshidandy/pokemon-selector.git
cd pokemon-selector

# ローカルサーバーの起動
python -m http.server 8000

# ブラウザでアクセス
open http://localhost:8000/
```

### 開発環境セットアップ
```bash
# 依存関係のインストール
pip install -r requirements.txt

# データの手動更新
python scripts/fetch_pokemon_data.py
python scripts/fetch_pokemon_detail.py
```
